282. Expression Add Operators

Hard

Given a string that contains only digits 0-9 and a target value, return all possibilities to add binary operators (not unary) +, -, or * between the digits so they evaluate to the target value.

Example 1:

Input: _num_ = “123”, target = 6
Output: [“1+2+3”, “123”]

Example 2:

Input: _num_ = “232”, target = 8
Output: [“23+2", "2+32”]

Example 3:

Input: _num_ = “105”, target = 5
Output: [“1*0+5”,“10-5”]

Example 4:

Input: _num_ = “00”, target = 0
Output: [“0+0”, “0-0”, “0*0”]

Example 5:

Input: _num_ = “3456237490”, target = 9191
Output: []

Constraints:

  • 0 <= num.length <= 10
  • num only contain digits.

[Solution]

class Solution {
    public List<String> addOperators(String num, int target) {
        List<String> res = new ArrayList<>();
        helper(num, target, 0, 0, "", res);
        return res;
    }
    
    void helper(String num, int target, long diff, long sum, String aResult, List<String> res) {
        if (num.length() == 0) {
            if (sum == target) {
                res.add(aResult);
            }
            return;
        }
        
        
        for (int i = 1; i <= num.length(); i++) {
            String first = num.substring(0, i);
            if (first.length() > 1 && first.charAt(0) == '0') continue;
            String second = num.substring(i);
            long firstNum = Long.parseLong(first);
            if (aResult.isEmpty()) {
                helper(second, target, firstNum, sum + firstNum, first, res);
            } else {
                helper(second, target, firstNum, sum + firstNum, aResult + "+" + first, res);
                helper(second, target, -firstNum, sum - firstNum, aResult + "-" + first, res);
                helper(second, target, diff * firstNum, sum - diff + diff *firstNum, aResult + "*" + first, res);
            }
        }
    }
}
