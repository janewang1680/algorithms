51. N-Queens

Hard

The n-queens puzzle is the problem of placing n queens on an n x n chessboard such that no two queens attack each other.

Given an integer n, return all distinct solutions to the n-queens puzzle.

Each solution contains a distinct board configuration of the n-queens’ placement, where 'Q' and '.' both indicate a queen and an empty space, respectively.

Example 1:

Input: n = 4
Output: [[".Q…","…Q",“Q…”,"…Q."],["…Q.",“Q…”,"…Q",".Q…"]]
Explanation: There exist two distinct solutions to the 4-queens puzzle as shown above

Example 2:

Input: n = 1
Output: [[“Q”]]

Constraints:

  • 1 <= n <= 9

[Solution]

class Solution {
    
    HashSet<Integer> col = new HashSet<>();
    HashSet<Integer> diagonalleft = new HashSet<>();
    HashSet<Integer> diagonalright = new HashSet<>();
    List<List<String>> res = new ArrayList<>();
    char[] template;
    public List<List<String>> solveNQueens(int n) {       
        template = new char[n];
        Arrays.fill(template, '.');
        dfs(n, new ArrayList<String>());
        return res;
    }
    
    private void dfs(int n, List<String> ans) {
        
        if(ans.size() == n) {
            res.add(new ArrayList<>(ans));
            return;
        }
        
        for(int i = 0; i < n; i++) {
            
            if(col.contains(i)) continue;
            if(diagonalleft.contains(i - ans.size())) continue;
            if(diagonalright.contains(i + ans.size())) continue;
               
            col.add(i);
            diagonalleft.add(i - ans.size());
            diagonalright.add(i + ans.size());
            
            template[i] = 'Q';
            String str = new String(template);
            template[i] = '.';
            
            ans.add(str);
            
            dfs(n, ans);
               
               ans.remove(ans.size() - 1);
               col.remove(i);
               diagonalleft.remove(i - ans.size());
               diagonalright.remove(i + ans.size());
        }
    }
}
