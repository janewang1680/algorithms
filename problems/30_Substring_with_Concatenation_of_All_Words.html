30. Substring with Concatenation of All Words

Hard

You are given a string s and an array of strings words of the same length. Return all starting indices of substring(s) in s that is a concatenation of each word in words exactly once, in any order, and without any intervening characters.

You can return the answer in any order.

Example 1:

Input: s = “barfoothefoobarman”, words = [“foo”,“bar”]
Output: [0,9]
Explanation: Substrings starting at index 0 and 9 are “barfoo” and “foobar” respectively.
The output order does not matter, returning [9,0] is fine too.

Example 2:

Input: s = “wordgoodgoodgoodbestword”, words = [“word”,“good”,“best”,“word”]
Output: []

Example 3:

Input: s = “barfoofoobarthefoobarman”, words = [“bar”,“foo”,“the”]
Output: [6,9,12]

Constraints:

  • 1 <= s.length <= 104
  • s consists of lower-case English letters.
  • 1 <= words.length <= 5000
  • 1 <= words[i].length <= 30
  • words[i] consists of lower-case English letters.

[Solution]

class Solution {
    public List<Integer> findSubstring(String s, String[] words) {
        if (s == null || s.length() == 0 || words == null || words.length == 0) return new ArrayList<>();
        
        Map<String, Integer> dict = new HashMap<>();
        for (String word : words) dict.put(word, dict.getOrDefault(word, 0) + 1);
        
        List<Integer> res = new ArrayList<>();
        
        int len = words[0].length();
        int cnt = words.length;
        
        for (int i = 0; i <= s.length() - cnt * len; i++) {
            int total = 0;
            Map<String, Integer> map2 = new HashMap<>();
            for (int j = 0; j < cnt; j++) {
                String tmp = s.substring(i + j * len, i + (j+1) * len);
                if (!dict.containsKey(tmp)) break;
                map2.put(tmp, map2.getOrDefault(tmp, 0) + 1);
                if (map2.get(tmp) > dict.get(tmp)) break;
                total++;
            }
            if (total == cnt) res.add(i);
            
        }
        
        return res;
    }
}
