679. 24 Game

Hard

You have 4 cards each containing a number from 1 to 9. You need to judge whether they could operated through *, /, +, -, (, ) to get the value of 24.

Example 1:

Input: [4, 1, 8, 7]
Output: True
Explanation: (8-4) * (7-1) = 24

Example 2:

Input: [1, 2, 1, 2]
Output: False

Note:

  1. The division operator / represents real division, not integer division. For example, 4 / (1 - 2/3) = 12.
  2. Every operation done is between two numbers. In particular, we cannot use - as a unary operator. For example, with [1, 1, 1, 1] as input, the expression -1 - 1 - 1 - 1 is not allowed.
  3. You cannot concatenate numbers together. For example, if the input is [1, 2, 1, 2], we cannot write this as 12 + 12.

[Solution]

class Solution {
    public boolean judgePoint24(int[] nums) {
        List<Double> list = new ArrayList<>();
        for (int num : nums) list.add((double)num);
        
        return helper(list);
    }
    
    boolean helper(List<Double> nums) {
        if (nums.size() == 1) return Math.abs(nums.get(0) - 24) < 0.001;
        
        int n = nums.size();
        for (int i = 0; i < n; i++) {
            double a = nums.get(i);
            for (int j = i + 1; j < n; j++) {
                double b = nums.get(j);
                
                List<Double> left = new ArrayList<>();
                for (int k = 0; k < n; k++) {
                    if (k != i && k != j) {
                        left.add(nums.get(k));
                    }
                }
                
                List<Double> r = compute(a, b);
                for (double d: r) {
                    left.add(d);
                    if (helper(left)) return true;
                    left.remove(left.size() - 1);
                }
            }
        }
        
        return false;
    }
    
    List<Double> compute(double a, double b) {
        List<Double> res = new ArrayList<>();
        res.add(a+b);
        res.add(a-b);
        res.add(b-a);
        res.add(a*b);
        res.add(1.0 * a /b);
        res.add(1.0 * b/a);
        return res;
    }
}
