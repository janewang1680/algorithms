639. Decode Ways II

Hard

A message containing letters from A-Z is being encoded to numbers using the following mapping way:

‘A’ -> 1
‘B’ -> 2
…
‘Z’ -> 26

Beyond that, now the encoded string can also contain the character ‘*’, which can be treated as one of the numbers from 1 to 9.

Given the encoded message containing digits and the character ‘*’, return the total number of ways to decode it.

Also, since the answer may be very large, you should return the output mod 109 + 7.

Example 1:

Input: “*”
Output: 9
Explanation: The encoded message can be decoded to the string: “A”, “B”, “C”, “D”, “E”, “F”, “G”, “H”, “I”.

Example 2:

Input: “1*”
Output: 9 + 9 = 18

Note:

  1. The length of the input string will fit in range [1, 105].
  2. The input string will only contain the character ‘*’ and digits ‘0’ - ‘9’.

[Solution]

class Solution {
    public int numDecodings(String s) {
        if (s == null || s.length() == 0) return 0;
        int mod = (int) Math.pow(10, 9) + 7;
        
        int n = s.length();
        long[] dp = new long[n+1];
        dp[0] = 1;
        for (int i = 1; i <= n; i++) {
            char c = s.charAt(i-1);
            long prev = i > 0 ? dp[i-1] : 0;
            if (c == '*') {
                dp[i] = (dp[i] + 9 * prev) % mod;
                if (i > 1) {
                    if (s.charAt(i-2) == '*') {
                        dp[i] = (dp[i] + 15 * (i > 1 ? dp[i-2] : 0)) % mod;
                    } else if (s.charAt(i-2) == '1') {
                        dp[i] = (dp[i] + 9 * (i > 1 ? dp[i-2] : 0)) % mod;
                    } else if (s.charAt(i-2) == '2') {
                        dp[i] = (dp[i] + 6 * (i > 1 ? dp[i-2] : 0)) % mod;
                    }
                }
            } else if (c == '0') {
                if (i > 1) {
                    if (s.charAt(i-2) == '*') {
                        dp[i] = (dp[i] + 2 * (i > 1 ? dp[i-2] : 0)) % mod;
                    } else if (s.charAt(i-2) == '2' || s.charAt(i-2) == '1') {
                        dp[i] = (dp[i] + (i > 1 ? dp[i-2] : 0)) % mod;
                    }
                }
            } else if (c >= '1' && c <= '6') {
                dp[i] = (dp[i] + prev) % mod;
                if (i > 1) {
                    if (s.charAt(i-2) == '*') {
                        dp[i] = (dp[i] + 2 * (i > 1 ? dp[i-2] : 0)) % mod;
                    } else if (s.charAt(i-2) == '2' || s.charAt(i-2) == '1') {
                        dp[i] = (dp[i] + (i > 1 ? dp[i-2] : 0)) % mod;
                    }
                }
            } else {
                dp[i] = (dp[i] + prev) % mod;
                if (i > 1) {
                    if (s.charAt(i-2) == '*') {
                        dp[i] = (dp[i] + (i > 1 ? dp[i-2] : 0)) % mod;
                    } else if (s.charAt(i-2) == '1') {
                        dp[i] = (dp[i] + (i > 1 ? dp[i-2] : 0)) % mod;
                    }
                }
            }
        }
        
        return (int) dp[n];
    }
}
