218. The Skyline Problem

Hard

A city’s skyline is the outer contour of the silhouette formed by all the buildings in that city when viewed from a distance. Given the locations and heights of all the buildings, return the skyline formed by these buildings collectively.

The geometric information of each building is given in the array buildings where buildings[i] = [lefti, righti, heighti]:

  • lefti is the x coordinate of the left edge of the ith building.
  • righti is the x coordinate of the right edge of the ith building.
  • heighti is the height of the ith building.

You may assume all buildings are perfect rectangles grounded on an absolutely flat surface at height 0.

The skyline should be represented as a list of “key points” sorted by their x-coordinate in the form [[x1,y1],[x2,y2],...]. Each key point is the left endpoint of some horizontal segment in the skyline except the last point in the list, which always has a y-coordinate 0 and is used to mark the skyline’s termination where the rightmost building ends. Any ground between the leftmost and rightmost buildings should be part of the skyline’s contour.

Note: There must be no consecutive horizontal lines of equal height in the output skyline. For instance, [...,[2 3],[4 5],[7 5],[11 5],[12 7],...] is not acceptable; the three lines of height 5 should be merged into one in the final output as such: [...,[2 3],[4 5],[12 7],...]

Example 1:

Input: buildings = [[2,9,10],[3,7,15],[5,12,12],[15,20,10],[19,24,8]]
Output: [[2,10],[3,15],[7,12],[12,0],[15,10],[20,8],[24,0]]

Example 2:

Input: buildings = [[0,2,3],[2,5,3]]
Output: [[0,3],[5,0]]

Constraints:

  • 1 <= buildings.length <= 104
  • 0 <= lefti < righti <= 231 - 1
  • 1 <= heighti <= 231 - 1
  • buildings is sorted by lefti in non-decreasing order.

[Solution]

class Solution {
    public List<List<Integer>> getSkyline(int[][] buildings) {
       List<List<Integer>> res = new ArrayList<>();
        if (buildings == null || buildings.length == 0) return res;
        
        return getSkyline(buildings, 0, buildings.length - 1);
    }
    
    List<List<Integer>> getSkyline(int[][] buildings, int l, int u) {
        List<List<Integer>> res = new ArrayList<>();
        if (l > u) return res;
        if (l == u) {
            List<Integer> list = new ArrayList<>();
            list.add(buildings[l][0]);
            list.add(buildings[l][2]);
            res.add(list);
            
            list = new ArrayList<>();
            list.add(buildings[l][1]);
            list.add(0);
            res.add(list);
            
            return res;
        }
        
        int mid = l + (u - l)/2;
         List<List<Integer>> lefts = getSkyline(buildings, l, mid);
        List<List<Integer>> rights = getSkyline(buildings, mid+1, u);
        
        int h1 = 0, h2 = 0, prev = Integer.MIN_VALUE;
        int i = 0, j = 0;
        while (i < lefts.size() && j < rights.size()) {
            List<Integer> left = lefts.get(i);
            List<Integer> right = rights.get(j);
            
            List<Integer> list = new ArrayList<>();
            
            if (left.get(0) < right.get(0)) {
                h1 = left.get(1);
                int max = Math.max(h1, h2);
                
                list.add(left.get(0));
                list.add(max);
                
                i++;
            } else if (left.get(0) > right.get(0)) {
                h2 = right.get(1);
                int max = Math.max(h1, h2);
                
                list.add(right.get(0));
                list.add(max);
                
                j++;
            } else {
                h1 = left.get(1);
                h2 = right.get(1);
                int max = Math.max(h1, h2);
                list.add(right.get(0));
                list.add(max);
                
                i++;
                j++;
            }
            
            if (prev == Integer.MIN_VALUE || prev != list.get(1)) res.add(list);
            prev = list.get(1);
        }
        
        while (i < lefts.size()) res.add(lefts.get(i++));
        while (j < rights.size()) res.add(rights.get(j++));
        
        return res;
    }
}
