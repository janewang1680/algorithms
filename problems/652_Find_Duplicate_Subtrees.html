652. Find Duplicate Subtrees

Medium

Given the root of a binary tree, return all duplicate subtrees.

For each kind of duplicate subtrees, you only need to return the root node of any one of them.

Two trees are duplicate if they have the same structure with the same node values.

Example 1:

Input: root = [1,2,3,4,null,2,4,null,null,4]
Output: [[2,4],[4]]

Example 2:

Input: root = [2,1,1]
Output: [[1]]

Example 3:

Input: root = [2,2,2,3,null,3,null]
Output: [[2,3],[3]]

Constraints:

  • The number of the nodes in the tree will be in the range [1, 10^4]
  • -200 <= Node.val <= 200

[Solution]

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public List<TreeNode> findDuplicateSubtrees(TreeNode root) {
        Map<String, TreeNode> map = new HashMap<>();
        Set<TreeNode> res = new HashSet<>();
        helper(root, map, res);
        return new ArrayList<>(res);
    }
    
    String helper(TreeNode root, Map<String, TreeNode> map, Set<TreeNode> res) {
        if (root == null) return null;
        
        String left = helper(root.left, map, res);
        String right = helper(root.right, map, res);
        
        String hash = "left:" + left + "," + String.valueOf(root.val) + ",right:" + right;
        if (!map.containsKey(hash)) map.put(hash, root);
        else {
            res.add(map.get(hash));
        }
        
        return hash;
    }
}
