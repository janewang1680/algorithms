564. Find the Closest Palindrome

Hard

Given an integer n, find the closest integer (not including itself), which is a palindrome.

The ‘closest’ is defined as absolute difference minimized between two integers.

Example 1:

Input: “123”
Output: “121”

Note:

  1. The input n is a positive integer represented by string, whose length will not exceed 18.
  2. If there is a tie, return the smaller one as answer.

[Solution]

class Solution {
    public String nearestPalindromic(String n) {
        int len = n.length();
        Set<String> set = new HashSet<>();
        set.add(String.valueOf(((int)Math.pow(10, len)) + 1));
        set.add(String.valueOf((int)Math.pow(10, len-1) - 1)); 
        
        int half = Integer.parseInt(n.substring(0, (len+1)/2));
        for (int i = -1; i <= 1; i++) {
            String tmp = String.valueOf(half+i);
            set.add(tmp + reverse(tmp, len));
        }
        set.remove(n);
        
        long min = Integer.MAX_VALUE;
        String minStr = "";
        for (String str: set) {
            long diff = Math.abs(Long.parseLong(str) - Long.parseLong(n));
            if (diff < min) {
                min = diff;
                minStr = str;
            } else if (diff == min && Long.parseLong(str) < Long.parseLong(n)) {
                minStr = str;
            }
        }
        
        return minStr;
    }
    
    String reverse(String str, int len) {
        if (len % 2 == 1) {
            StringBuilder builder = new StringBuilder(str.substring(0, str.length() - 1));
            return builder.reverse().toString();
        } else {
            StringBuilder builder = new StringBuilder(str);
            return builder.reverse().toString();
        }
    }
}
