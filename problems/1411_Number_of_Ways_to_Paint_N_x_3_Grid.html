1411. Number of Ways to Paint N × 3 Grid

Hard

You have a grid of size n x 3 and you want to paint each cell of the grid with exactly one of the three colours: Red, Yellow or Green while making sure that no two adjacent cells have the same colour (i.e no two cells that share vertical or horizontal sides have the same colour).

You are given n the number of rows of the grid.

Return the number of ways you can paint this grid. As the answer may grow large, the answer must be computed modulo 10^9 + 7.

Input: n = 1
Output: 12

Example 2:

Input: n = 2
Output: 54

Example 3:

Input: n = 3
Output: 246

Example 4:

Input: n = 7
Output: 106494

Example 5:

Input: n = 5000
Output: 30228214

Constraints:

  • n == grid.length
  • grid[i].length == 3
  • 1 <= n <= 5000

[Solution]

class Solution {
    public int numOfWays(int n) {
        // every line can either have two colors or three colors: 121 and 123.
        // for two color, it can be 121 131 212 232 313 323
        // for three color, it can be 123 132 213 231 312 321
        // first row have 12 ways
        // next row have:
        // for 121, it can be 212 213 232 312 313 
        // for 123, it can be 212 231 232 312
        
        int mod = (int) Math.pow(10, 9) + 7;
        
        long a121 = 6, a123 = 6;
        for (int i = 2; i <= n; i++) {
            long b121 = a121 * 3 + a123 * 2;
            long b123 = a121 * 2 + a123 * 2;
            a121 = b121 % mod;
            a123 = b123 % mod;
        }
        
        return (int) (a121 + a123) % mod;
    }
}
