772. Basic Calculator III

Hard

Implement a basic calculator to evaluate a simple expression string.

The expression string contains only non-negative integers, '+', '-', '*', '/' operators, open '(' and closing parentheses ')' and empty spaces ' '. The integer division should truncate toward zero.

You may assume that the given expression is always valid. All intermediate results will be in the range of [-231, 231 - 1].

Example 1:

Input: s = “1 + 1”
Output: 2

Example 2:

Input: s = " 6-4 / 2 "
Output: 4

Example 3:

Input: s = “2*(5+5*2)/3+(6/2+8)”
Output: 21

Example 4:

Input: s = “(2+6* 3+5- (3*14/7+2)*5)+3”
Output: -12

Example 5:

Input: s = “0”
Output: 0

Constraints:

  • 1 <= s <= 104
  • s consists of digits, '+', '-', '*', '/', '(', ')' and ' '.
  • s is a valid expression.

Follow up: Could you solve the problem without using built-in library functions?

[Solution]

class Solution {
    public int calculate(String s) {
        if (s == null || s.length() == 0) return 0;
        
        int[] res = helper(s, 0);
        return res[0];
    }
    
    int[] helper(String s, int start) {
        Stack<Character> operators = new Stack<>();
        Stack<Long> operands = new Stack<>();
        
        int i = start;
        while (i < s.length()) {
            char c = s.charAt(i);
            if (c == ' ') {
                i++;
                continue;
            } else if (c == '+' || c == '-') {
                while (!operators.isEmpty()) {
                    operands.push(compute(operators.pop(), operands.pop(), operands.pop()));
                }
                operators.push(c);
                i++;
            } else if (c == '*' || c == '/') {
                while (!operators.isEmpty() && operators.peek() != '+' && operators.peek() != '-') {
                    operands.push(compute(operators.pop(), operands.pop(), operands.pop()));
                }
                operators.push(c);
                i++;    
            } else if (c == '(') {
                int[] tmp = helper(s, i+1);
                operands.push((long) tmp[0]);
                i = tmp[1] + 1;
            } else if (c == ')') {
                break;
            } else {
                int j = i;
                while (j < s.length() && isDigit(s.charAt(j))) j++;
                long num = Long.parseLong(s.substring(i, j));
                operands.push(num);
                i = j;
            }
        }
        
        while (!operators.isEmpty()) {
            operands.push(compute(operators.pop(), operands.pop(), operands.pop()));
        }
        
        return new int[]{(int) ((long) operands.peek()), i};
    }
    
    boolean isDigit(char c) {
        return c >= '0' && c <= '9';
    }
    
    long compute(char c, long a, long b) {
        if (c == '+') return a+ b;
        else if (c == '-') return b - a;
        else if (c == '*') return a * b;
        else return b / a;
    }
}
